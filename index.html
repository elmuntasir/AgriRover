About the Project

AgriRover is an autonomous agricultural rover designed to reduce unnecessary herbicide and pesticide usage through AI-assisted precision farming. The rover surveys crop fields using an onboard camera (OV2640), detects weeds, pest nests, and unhealthy crops using a YOLOv8 computer vision model, and sends captured images and sensor data to a laptop-based control room via Wi-Fi. A field map of detected anomalies is generated in real time, and a route optimization module (TSP/VRP solver) minimizes travel distance, battery consumption, and chemical usage before executing targeted weed cutting and spot spraying.
