# components/pic/

Put every component photo here.

- Use a short, lowercase, hyphenated filename that matches the component,
  e.g. `esp32-s3.jpg`, `neo6m-gps.png`, `l298n-driver.jpg`.
- JPG or PNG both work. Keep each photo under ~500KB (resize/compress first
  so the page loads fast).
- Square-ish product photos on a plain background look the most consistent
  in the card layout, but any clear photo of the part is fine.

Once a photo is here, reference it from `components/component-template.js`
using the path `components/pic/your-file-name.jpg` in that component's
`pic` field.

If a `pic` field is left empty, or the file can't be found, the component
list page will automatically show a placeholder icon instead — so it's
fine to add the data first and drop the photo in later.
